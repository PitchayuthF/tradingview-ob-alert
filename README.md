# TradingView OB Alert Project

## Overview
The TradingView OB Alert project is designed to facilitate easy integration with TradingView alerts for various trading strategies. The goal is to create a seamless experience for users to receive alerts based on their specific criteria, utilizing TradingView's powerful charting capabilities along with the OB (Order Block) trading strategy.

## Features
- **Real-Time Alerts**: Instant notifications for trading opportunities based on specified conditions.
- **Line OA Integration**: Integration with Line OA for sending alerts directly to your Line app.
- **Customizable Settings**: Users can tailor their alert conditions based on their trading strategies.

## Installation
To install the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/PitchayuthF/tradingview-ob-alert.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tradingview-ob-alert
   ```
3. Install the necessary packages (if any) using your preferred package manager.

## Usage
To use the TradingView OB Alert, you will need to:
1. Set up a TradingView account and create your desired trading strategy.
2. Configure the alert conditions within TradingView, ensuring they match your strategy.
3. Set up the Line OA token and endpoint to receive alerts on your mobile.
4. Test the integration to confirm that alerts are being sent appropriately.
